# 📚 Library Management System

A web-based Library Management System built with C# and ASP.NET.

## Features
- User login and authentication
- Add, search, and manage books
- Issue and return books
- Forgot password functionality

## Tech Stack
- C# (ASP.NET Web Forms)
- SQL Server (Database)
- HTML/CSS (Frontend)

## How to Run
1. Open `Library managemant Amar.sln` in Visual Studio
2. Set up the database connection in `DBconnect.cs`
3. Run the project (F5)
